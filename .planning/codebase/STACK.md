# Tech Stack

**Language & Runtimes**
- Backend: Python 3.x
- Frontend: JavaScript (Node.js, React 19)

**Backend Frameworks & Libraries**
- Web Framework: FastAPI (Uvicorn, WebSockets)
- ML & Deep Learning: PyTorch (LSTM Autoencoder), PyTorch Geometric (GAT Localiser)
- Classical ML: XGBoost (TreeSHAP for explainability)
- Hydraulic Simulation: WNTR (Water Network Tool for Resilience - EPANET wrapper)
- Database: SQLite

**Frontend Frameworks & Libraries**
- Core: React 19, Vite
- Styling: Tailwind CSS (Custom "Lunar Observatory" Stitch Design System)
- Animations: Framer Motion
- Visualizations: D3.js (Network Graph), Chart.js (Sensor Series)
- Icons: Lucide-React, Material Symbols

**Tools & Build**
- Python Environment: virtualenv/venv
- Model Packaging: ONNX (with validation)
